# Prompt: Add Informative Sections to Home Page

## Context

Add educational and trust-building content sections below the main price comparison feature on the home page of a precious metals price comparison website (OunceTracker).

## Requirements

### Content Sections Needed:

1. **Why Gold & Silver Matter**
   - Brief history of precious metals as currency
   - Store of value and inflation hedge
   - Portfolio diversification benefits
   - Tangible asset advantages
   - Long-term wealth preservation

2. **Trusted Mints & Manufacturers**
   - Major mints: PAMP Suisse, Royal Canadian Mint, US Mint, Perth Mint, Valcambi
   - What makes a mint reputable
   - Certification and hallmarks
   - Geographic distribution

3. **The Economic Role of Precious Metals**
   - Central bank reserves
   - Industrial uses (especially silver)
   - Market indicators
   - Historical performance during crises
   - Supply and demand dynamics

4. **About OunceTracker / Why Trust Us**
   - Mission: transparent price comparison
   - Real-time data updates
   - Independent comparison (no dealer affiliations)
   - Data sources and update frequency
   - Commitment to accuracy

5. **Understanding Bullion Pricing** (Optional)
   - Spot price vs. dealer premiums
   - Factors affecting prices
   - Premiums by product type
   - Tips for buying

### Design Requirements:

- **Layout**: Mixed approach - different sections can use different layouts (cards, full-width, accordion) based on what works best for each
- **Content Type**: Static content (no dynamic updates needed)
- **Interactive Elements**: Include interactive elements like expandable FAQs, calculators, or collapsible sections where appropriate
- **CTAs**: No call-to-action buttons needed
- **Responsive**: Mobile-first design approach

### Visual Elements:

- Icons for each section (gold coin, mint building, chart, shield/trust badge)
- Subtle gold/silver color accents
- Professional but accessible tone
- Data-driven with stats where relevant
- Educational and trust-building

### Placement:

- Below the main price comparison table
- Above the footer
- Natural reading flow

### Technical Notes:

- Use React components with TypeScript
- Follow existing code patterns in the project
- Use Tailwind CSS for styling
- Ensure accessibility (ARIA labels, semantic HTML)
- Mobile-first responsive design
